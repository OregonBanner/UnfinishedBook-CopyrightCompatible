## 工具链

解包使用的版本为 `未竟之书-1.0.20260217-pc`。

解包后的结果按原样提供。

- [Shizmob/rpatool](https://github.com/Shizmob/rpatool)
  - 用于解包原始的 `game/archive.rpa` 文件。
- [CensoredUsername/unrpyc](https://github.com/CensoredUsername/unrpyc)
  - 用于反编译解包后的 `.rpyc` 文件。

为节省空间，本项目不再包含原始文件。
